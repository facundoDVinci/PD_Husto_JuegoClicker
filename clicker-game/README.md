# Clicker Game

Welcome to the Clicker Game! This is a simple yet addictive game where players can earn points by clicking a button and use those points to purchase upgrades and increase their clicks per second.

## Features

- **Click to Earn Points**: Click the button to earn points.
- **Upgrades**: Use points to purchase upgrades that enhance your clicking power.
- **Clicks Per Second**: Invest points to gain automatic clicks over time.
- **Responsive Design**: The game is built with Bootstrap 5.3 to ensure it looks great on all devices.

## Getting Started

To run the Clicker Game locally, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd clicker-game
   ```

2. **Install Dependencies**:
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Open the Game**:
   Open `src/index.html` in your web browser to start playing.

## Usage

- Click the main button to earn points.
- Use the points to buy upgrades that will increase your click value.
- Purchase clicks per second to automate point earning.
